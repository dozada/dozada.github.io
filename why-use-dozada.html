Dozada

Open source POS platform for retailers

Why use Dozada

Dozada's built to resolve issues from old POS systems hard to scale and customize.

Multi device POS

By using React Native on mobile and ElectronJs on desktop, Dozada provide a platform for developer easily to customize the layout and choose whatever OS they want to use for their system.

Database scalable

CouchDB's chosen because Seamless multi-master sync, that scales from Big Data to Mobile, with an Intuitive HTTP/JSON API and designed for Reliability.

API for external apps

We plan to build a API gateway so that developer can use it to build a website or customer loyalty apps

Integrate to ERP sytems

By using Dozada platform you have the right to choose whatever ERP system you want, Dozada plan to integrate with almost popular ERP system.
